📲 Add to Home Screen

On iPhone or iPad, open this page in Safari, then:

  1. Tap the Share button (the square with an up-arrow).
  2. Scroll down and tap Add to Home Screen.
  3. Tap Add. The app icon appears on your home screen.

🏆 Win Leaderboard

Wins are tracked per saved profile. Win a game with a player whose name matches a saved profile and it counts here.

Export saves all your players, commanders & wins to a file. Import merges that file into this device — safe to run more than once (it never double-counts wins).

Start New Game?

Choose how to start the next game:

  • Continue with Current Profiles — keeps names, colors, and commanders. Only life totals and counters reset.
  • Fresh New Game — also clears player names, colors, and commander setups (your saved profiles are kept). Table orientations stay the same.
